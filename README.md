**Versión de Java Recomendada**
Aunque es probable que el proyecto se ejecute correctamente en versiones posteriores de Java (como Java 11, 17 o superior), recomendamos fervientemente el uso de JDK 8 para garantizar la compatibilidad total y evitar posibles errores de ejecución o problemas de dependencia que pudieran surgir debido a diferencias en las versiones de Java
    _Si experimentas cualquier problema o error inesperado al ejecutar el proyecto, por favor, verifica y asegúrate de estar utilizando Java 8 (o el JRE/JDK 1.8.x)._
